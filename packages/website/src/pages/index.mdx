## include

next you want to include the webring on your personal website! it's
super easy, just add this script tag to the very end of your website's
body:

```js
<script src="https://pagering.gideon.sh/embed.js" defer></script>
```

then, wherever you want the webring link to appear in your page, add:

```html
<pagering-link></pagering-link>
```

<Image
  src={linkScreenshot}
  alt="screenshot of the webring link, light background"
/>

there are also some extra options you can configure:

- **theme:** the colour of the link - if your website always has a light or dark background, set this to
  'light' or 'dark'
  <small>"light" | "dark" | "system" (default: system)</small>
- **class/style:** add spacing, positioning, or other styles

```html
<!-- for example -->
<pagering-link theme="dark" style="margin-top: 8px"></pagering-link>
```

  alt="screenshot of the webring link, dark background"

**GET /api/v1/members:** get all members of the webring

```http
GET /api/v1/members

200 OK
[
  {
    "id": "acon",
    "name": "acon",
    "url": "https://aconlin.com",
    "buttonUrl": "/buttons/acon.gif"
  },
  ...
]
```

**GET /api/v1/members/[id]:** get a specific member by id, and the previous and next members

```http
GET /api/v1/members/acon

200 OK
{
  "current": {
    "id": "acon",
    "name": "acon",
    "url": "https://aconlin.com",
    "buttonUrl": "/buttons/acon.gif"
  },
  "prev": { ... },
  "next": { ... }
}
```

**GET /api/v1/embed:** get all members, and the current, previous, and next members for the current site.

<small>
  If you're calling this from a server, set your 'Origin' header to your site's
  URL (including http:// or https://)
</small>

```http
GET /api/v1/embed
# provided by your browser :)
Origin: https://aconlin.com

200 OK
{
  "current": {

    "url": "https://aconlin.com",
```
